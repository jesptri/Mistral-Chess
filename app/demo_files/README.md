# Demo Files Directory

This directory contains demonstration PGN files that users can load without uploading their own files.

## Adding Demo Files

To add a demo file:
1. Place your PGN file in this directory
2. Name it `Mazzella-Espinoux.pgn` (this is the default file that will be loaded)
3. You can add multiple files if needed

## Demo File Format

The demo file should be a standard PGN (Portable Game Notation) file, such as:
- Exported games from Chess.com
- Lichess game exports
- Any standard chess PGN format

## Example PGN Content

